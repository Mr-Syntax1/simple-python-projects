# Simple Python Projects

A collection of simple, beginner-friendly Python projects. These apps are built with pure Python, focusing on core programming concepts and practical utility — without relying on external frameworks.

---

## 🎯 Purpose of This Repository

This is a personal front-end lab for practicing **Python fundamentals** — by building small but real applications.  
Focuses on:
- Core Python syntax and logic
- Data structures and algorithms
- Common scripting tasks
- Object-Oriented Programming (OOP) principles
- Building small, self-contained utilities
